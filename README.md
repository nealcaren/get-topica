# get-topica

Landing page for [topica](https://github.com/nealcaren/topica) — a fast,
all-purpose topic-modeling library for Python.

Live at **https://nealcaren.github.io/get-topica/**.

Single self-contained `index.html` (inline CSS/JS, no external requests). The
project's documentation lives separately at
<https://nealcaren.github.io/topica/>.
