# get-topica

Landing pages for [topica](https://github.com/nealcaren/topica) — a fast,
all-purpose topic-modeling library for Python.

Live at **https://nealcaren.github.io/get-topica/**.

Each page is a single self-contained HTML file (inline CSS/JS, no external
requests):

- **`index.html`** — the home page: a replayed terminal session that walks a
  full topica workflow (fit, covariate effects, best-k, ensembles, embeddings)
  with inline ASCII figures.
- **`site/index.html`** — the full marketing site, opening with the animated
  "words into bins" hero, the model roster, speed benchmarks, and more.
- **`bins/index.html`** — the standalone "words into bins" illustration.
- **`terminal/index.html`** — redirects to the home page (the terminal used to
  live here).

The project's documentation lives separately at
<https://nealcaren.github.io/topica/>.
